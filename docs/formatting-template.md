# 清华大学出版社格式调整模板

## 一、标题层级格式（5级）

```
# 第1章 ××××                                    # 一级标题：三号黑体，居中
## 1.1 ××××                                   # 二级标题：小三号黑体，顶格
### 1.1.1 ××××                              # 三级标题：四号黑体，顶格
#### 1. ××××                                   # 四级标题：小四号黑体，前空两格
##### 1) ××××                                  # 五级标题：五号黑体，前空两格
```

## 二、正文格式

```
正文采用五号字体（中文为宋体，英文为Time New Roman）
```

## 三、第一次出现的术语格式

```
# 第一次出现的关键术语需要：
# 1. 中文术语加黑体
# 2. 英文缩写给出全称（夹注）

例如：Transformer 是一种基于自注意力机制（Self-Attention）的深度学习架构。

# 术语表中应包含：
# Transformer (Transformer) -变换器
# Self-Attention (Self-Attention) -自注意力机制
# KV Cache (Key-Value Cache) -键值缓存
```

## 四、公式格式

```
# 公式居中，右侧编号
公式应写成：

P = K × F / (1000 × η)                              (1-1)

式中：P——电动机功率（单位：kW）；
      K——安全因数，K = 1.2 ~ 1.5；
      F——链传动限力式辊子输送机积放状态下链条的最大牵引力（单位：N）。
```

# 第1章 推理优化概述

> 本章重点：理解推理优化的本质和重要性

## 1.1 什么是推理

**推理（Inference）**是指已经训练好的模型对新的输入进行处理并产生输出的过程。

### 1.1.1 训练与推理的区别

训练（Training）和推理的主要区别包括：

| 维度 | 训练 | 推理 |
|------|------|------|
| 计算模式 | 前向+反向 | 仅前向 |
| 资源消耗 | 高（需要GPU集群）| 相对较低 |
| 延迟要求 | 无实时要求 | 有实时性要求 |

### 1.1.2 推理的重要性

如图1-1所示，推理是AI模型产生实际价值的关键环节。

## 1.2 推理优化的目标

### 1.2.1 主要指标

推理优化的核心指标包括：

1) **延迟（Latency）**：TPOT、TTFT
2) **吞吐（Throughput）**：tokens/s
3) **成本（Cost）**：$/token

## 本章小结

本章介绍了推理的基本概念和优化目标。
