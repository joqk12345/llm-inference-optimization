# Word文档导入说明

## 已生成的文件

- `LLM推理优化实战_清华格式.html` - 符合清华大学出版社格式要求的HTML文档

## 导入Word方法

### 方法一：直接打开（推荐）

1. 打开Microsoft Word
2. 文件 → 打开
3. 将文件类型选择为"所有文件(*.*)"
4. 选择 `LLM推理优化实战_清华格式.html`
5. Word会自动转换格式

### 方法二：另存为docx

1. 按方法一打开文件
2. 文件 → 另存为
3. 选择"Word文档(*.docx)"格式
4. 保存
